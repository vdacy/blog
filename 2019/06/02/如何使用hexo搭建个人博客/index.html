如何使用hexo搭建个人博客

Posted on 2019-06-02

简介

现如今，越来越多的技术人员都搭建了自己的技术博客。此次就借用github的服务器，使用hexo做为博客搭建的框架，将博客托管到 Github Pages 上，他支持配置自定义域名，可以在该页面中绑定，当放域名时，会自动映射到 Github 的个人博客上。

hexo安装

安装hexo必须需要在电脑上 *先安装 Node.js、Git *，后才能安装 hexo

  • Node.js 安装：

我使用的是 win10 系统进行安装的，安装 Node.js 有个诀窍,那就是 闭上眼 ，一直点击 一步(next)就可以了，当然安装路径根据自己需要进行修改。

下载链接：Node: https://nodejs.org/en/download/

  • Git 安装：

此处也是 win10 进行安装的，由于已经安装过了，需要安装的朋友可以参考百度。

下载链接：Git：https://git-scm.com/downloads

必备的 NodeJs、Git 安装后，接下来就可以进行 hexo 的安装了

npm install -g hexo-cli
